# Lucky Gashapon 2.0
Lucky Gachapon is a spin-based item collecting game. 
The player starts with 150 points and spends 10 per spin to earn Common, Rare, or Ultra Rare items. Items can be sold to keep spinning.
**New Game+ mode that challenges you to collect every item in the game as fast as possible.
